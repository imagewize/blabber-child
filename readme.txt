== Description ==

Blabber Child is a child theme of the Blabber WordPress theme. It includes additional functionality such as a toggle button to hide/show gambling advertisements.

== Installation ==

1. Upload the `blabber-child` folder to your `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Configure your site as you would with the Blabber parent theme

== Frequently Asked Questions ==

= Does this theme require a parent theme? =

Yes, this theme requires the Blabber theme to be installed.

= How does the gambling ads toggle work? =

The toggle button in the footer allows users to hide or show gambling advertisements. When clicked, it hides all elements with the class `code-block`. The setting is saved in a browser cookie for 365 days.
